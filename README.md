**Personal Portfolio**
A responsive personal portfolio website built as part of the Harvard CS50x (2026) curriculum.

**🚀 Live Demo**
https://cs50-portfolio-three.vercel.app

**💡 Features**
1. Custom Design: Built from scratch with pure HTML, CSS, and vanilla JavaScript.
2. Interactive DOM: Features custom event listeners and dynamic elements.
3. Responsive Layout: Uses modern CSS padding and layout techniques to look great on desktop and mobile.
4. Continuous Deployment: Automatically deployed and hosted via Vercel.

**🛠️ Technologies Used**
HTML5
CSS3
JavaScript

**💻 Local Setup**
Since this is a static frontend site, no backend server is required to run it locally.
1. Clone this repository.
2. Double-click index.html to open it directly in your web browser.
