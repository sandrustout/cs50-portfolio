Personal Portfolio
A responsive personal portfolio website built as part of the Harvard CS50x (2026) curriculum.

🚀 Live Demo
https://cs50-portfolio-three.vercel.app

💡 Features
Custom Design: Built from scratch with pure HTML, CSS, and vanilla JavaScript.

Interactive DOM: Features custom event listeners and dynamic elements.

Responsive Layout: Uses modern CSS padding and layout techniques to look great on desktop and mobile.

Continuous Deployment: Automatically deployed and hosted via Vercel.

🛠️ Technologies Used
HTML5

CSS3

JavaScript

💻 Local Setup
Since this is a static frontend site, no backend server is required to run it locally.

Clone this repository.

Double-click index.html to open it directly in your web browser.
